Как создать лекцию:
1) Скопировать предыдущую папку, поменяв цифру на 1 больше;
2) Техать все в файле lecture.tex, там имеются всякие секции, а именно
\begin{lecture}
\begin{lecSection}
\begin{lecSubsection}

Самое главное - указывать в путях для добавляемых файлов переменную \myFolder.
Так, новый cpp файл должен иметь путь \cppfile{\myFolder/main}{....} (указывается без расширения)
Нельзя писать \cppfile{main}{...} !!!
Если не выполнить это требование, большой проект не будет собираться

3) Чтобы собрать лекцию, достаточно выполнить 'make' в её каталоге. Иногда latex может баговать,
поэтому можно использовать 'make twice' (соберет дважды). ИСПРАВЛЕНО: теперь по умолчанию собирает дважды. Тем не менее, нужны тесты!
4) Чтобы очистить папку от логов\gz\прочих прелестей, используется 'make clean'.
Он НЕ удаляет pdf файл. pdf файлы можно удалить, введя 'make clean_all'
5) Чтобы собрать большой проект, надо воспользоваться 'make together' из папки informatics.
Опять же, временные файлы чистятся 'make clean'; 'make together_twice' соберет финальный проект дважды (два раза применит цель 'all'. В цель 'all' уже включено сборка дважды, так что едва ли это правило стоит осавлять. Возможно, уберу его в следующем коммите).
